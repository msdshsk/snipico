# Snipico

軽量画像クロップツール - アイコン作成に特化した Electron アプリ

![Snipico](build/icon.ico)

## 機能

### クロップモード
- **中心クロップ (1:1)** - 画像中心から正方形で切り出し、サイズはスライダーで調整
- **フリー選択** - 任意の範囲をドラッグで選択
- **固定サイズ** - ピクセル単位で指定したサイズで切り出し

### 出力形式
- **PNG / JPEG / WebP** - 通常の画像形式
- **Windows ICO** - マルチサイズアイコン (16, 24, 32, 48, 64, 128, 256px)
- **個別 PNG** - 各サイズを別ファイルで出力

### その他の機能
- **角丸** - 切り出し画像に角丸を適用（円形も可）
- **ズーム/パン** - マウスホイールで拡大、ドラッグで移動
- **ICO 閲覧モード** - 既存の ICO ファイルの内容を確認・抽出
- **ドラッグ&ドロップ** - 画像ファイルを直接ドロップして読み込み

## インストール

```bash
npm install
```

## 開発

```bash
npm start
```

## ビルド

```bash
npm run build
```

`dist/Snipico 1.0.0.exe` にポータブル版が生成されます。

## 依存関係

- [Electron](https://www.electronjs.org/) - デスクトップアプリフレームワーク
- [Sharp](https://sharp.pixelplumbing.com/) - 画像処理ライブラリ
